# Prediction-sur-les-maladies-cardiovasculaires
Basé sur des modeles de machines learning pour predire les cas de maladie cardiovasculaire.
On entraine un ensemble de données basé sur un data set "pima indian" existant, sur plusieurs modeles 
de ML puis les comparés en fonction de plusieurs metriques ou kpi d'evaluations pour avoir le modele 
le plus sure et le plus precis pour la problematique.
